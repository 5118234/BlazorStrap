# BlazorStrap

Port of https://reactstrap.github.io to Blazor

## Bootstrap 4.1

**IMPORTANT** Blazor 0.3.0 ships with Bootstrap 4.0. BlazorStrap uses Bootstrap 4.1. Replace bootstrap.min.css in wwwroot/css/bootstrap folder
