Buttons

Currently Blazor doesnt allow for components to be named the same name as html elements. See: https://github.com/aspnet/Blazor/issues/647. Therefore the component is currently named BlazorButton. 
          <BlazorButton Color=Color.Primary>primary</BlazorButton>
          <BlazorButton Color=Color.Secondary>secondary</BlazorButton>
          <BlazorButton Color=Color.Success>success</BlazorButton>
          <BlazorButton Color=Color.Info>info</BlazorButton>
          <BlazorButton Color=Color.Warning>warning</BlazorButton>
          <BlazorButton Color=Color.Danger>danger</BlazorButton>
          <BlazorButton Color=Color.Light>light</BlazorButton>
          <BlazorButton Color=Color.Dark>Dark</BlazorButton>

Supplying on Click Handler

          <BlazorButton OnClick="@onclick">Click Me</BlazorButton>
          @functions {
              void onclick(UIMouseEventArgs e)
              {
                  Console.WriteLine("Button was clicked!");
              }
          }

Button tags

Link 
          <BlazorButton ButtonType=ButtonType.Link Href="#">Link</BlazorButton>
          <BlazorButton ButtonType=ButtonType.Button>Button</BlazorButton>
          <BlazorButton ButtonType=ButtonType.Input Value="Input" />
          <BlazorButton ButtonType=ButtonType.Submit Value="Submit" />
          <BlazorButton ButtonType=ButtonType.Reset Value="Reset" />

Primary Link Link
