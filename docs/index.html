Blazor Strap

A port of react-strap to the Blazor Framework

Install

Package Manager

Install-Package BlazorStrap -Version 0.0.2

dotnet CLI

dotnet add package BlazorStrap --version 0.0.2

Add Bootsrap 4 to index.html

Add Bootstrap 4 stylesheet to index.html instead of bootstrap 3. No need to add bootstrap javascript file.

Tag Helpers

@addTagHelper *, BlazorStrap
React Strap will not work propertly with Blazor 0.2.0. Blazor 0.3.0 will fix this issue.
