Blazor Strap

Bootstrap 4 Components for Blazor Framework

Blazorous

If you want to manage your styles in code and use dynamic styles you can check out my other project: Blazorous. In the future I will show examples of mixing and matching BlazorStrap and Blazorous.

Install

Tag Helpers

@addTagHelper *, BlazorStrap
Blazor 0.3.0 ships with Bootstrap 4.0. BlazorStrap uses Bootstrap 4.1. Replace bootstrap.min.css in wwwroot/css/bootstrap folder
The interactive demos in this site will not work in this static site. To see the samples in action run the sample project locally.
